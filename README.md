# Royal Sailing

Royal Sailing is a lightweight project home for a premium sailing, charter, and maritime-experience surface in the Royal ecosystem.

This repository starts intentionally small: a static first page, a project brief, and enough structure to move toward booking, concierge, or content workflows without burying the idea under infrastructure.

## Current Status

- Repository initialized from an empty GitHub repo.
- Static landing page scaffolded in `index.html`.
- Project direction captured in `docs/project-brief.md`.

## Near-Term Direction

1. Decide the first commercial motion: charter inquiry, sailing club waitlist, itinerary planning, or content-led lead capture.
2. Add one clear call to action and a durable intake path.
3. Connect deployment only after the offer and destination are clear.

## Guardrails

- Keep secrets and customer data out of this repo.
- Prefer simple static hosting until there is a real workflow that needs a backend.
- Keep copy grounded in the actual offer; do not imply licensed services, ownership, or availability that has not been verified.
