# Project Brief

## Working Concept

Royal Sailing is a premium maritime experience brand surface. It can evolve into one of three practical directions:

- Charter and trip inquiry page
- Sailing lifestyle/content lead funnel
- Concierge itinerary planner for coastal trips

## First Useful Version

The first useful version should answer four questions quickly:

1. What kind of sailing experience is offered?
2. Where is it available?
3. Who is it for?
4. What should an interested visitor do next?

## Suggested Offer Shape

Start with a restrained inquiry flow:

- Desired region or marina
- Preferred date window
- Group size
- Experience type: sunset sail, day charter, lessons, private event, or custom
- Contact information

## Open Decisions

- Geographic focus
- Whether this is an owned-operation brand, affiliate/concierge model, or content funnel
- Booking/payment provider
- Deployment target

## Safety Notes

- Do not claim captain credentials, vessel availability, insurance coverage, or charter licensing without evidence.
- If this becomes an affiliate/concierge model, disclose that clearly.
- Keep payment collection separate until the service model is verified.
